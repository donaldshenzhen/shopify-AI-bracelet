# 冥想助手应用

一个沉浸式冥想应用，提供计时器、背景音乐和响应式界面，帮助用户进行专注的冥想练习。

## 功能特点

### 1. 全屏沉浸式体验
- 深色主题设计，创造专注的冥想环境
- 背景渐变和微妙动画效果
- 隐藏浏览器导航元素

### 2. 灵活的计时器功能
- 自定义冥想时长设置
- 预设时间按钮（5分钟、10分钟、15分钟等）
- 直观的进度环显示
- 播放/暂停和重置控制

### 3. 冥想音乐播放器
- 多种环境音乐选择
- 音量控制
- 音乐切换功能
- 加载状态指示

### 4. 设备适配
- 横屏模式偏好和提示
- 响应式设计，适配各种设备尺寸
- 触摸优化交互
- 高分辨率屏幕支持

## 技术栈

- React 18
- Vite
- CSS3 (响应式设计、动画效果)
- Web APIs (音频处理、屏幕方向)

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 键盘快捷键

- **空格键**: 播放/暂停计时器
- **R键**: 重置计时器
- **上箭头**: 增加音量
- **下箭头**: 减少音量

## 响应式设计

应用针对不同设备进行了优化：

- 桌面端: 完整功能体验
- 平板设备: 调整元素大小和间距
- 移动设备: 优化触摸体验，简化布局
- 横屏模式: 专为手表等小屏幕设备设计

## 无障碍支持

- 键盘导航
- 焦点样式
- 语义化HTML结构

## 性能优化

- 硬件加速
- 动画性能优化
- 事件处理防抖
- 错误边界和优雅降级

## 注意事项

1. 应用推荐在横屏模式下使用
2. 对于移动设备，建议禁用系统通知以获得更好的冥想体验
3. 音频文件需要在实际部署时提供有效的URL
