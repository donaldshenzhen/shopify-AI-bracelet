# PWA Icons

This directory contains the PWA application icons in various sizes:

- icon-72x72.png
- icon-96x96.png  
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

The icons represent the AI Bracelet Meditation app with a smart bracelet design and meditation waves.

To generate proper PNG icons from the SVG source, you can use tools like:
- Online SVG to PNG converters
- ImageMagick: `convert icon.svg -resize 192x192 icon-192x192.png`
- Figma or other design tools

For now, placeholder icons should be created manually or using online tools.
