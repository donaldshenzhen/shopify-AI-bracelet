# 视频背景文件夹

请将您的视频文件复制到此目录中：

## 需要的视频文件：
1. **forest-background.mp4** - 森林背景视频
2. **candlelight-background.mp4** - 烛光背景视频

## 视频要求：
- 格式：MP4
- 建议分辨率：1920x1080 或更高
- 时长：建议 10-30 秒循环播放
- 文件大小：建议小于 50MB

## 如何添加视频：
1. 从您的下载文件夹中找到对应的视频文件
2. 重命名为上述指定的文件名
3. 复制到这个 videos 文件夹中

完成后，应用将支持：
- 视频背景播放
- 左右滑动切换视频背景
- 保持原有的冥想音乐功能
